# BREAKING: Local Man Discovers Canva

A satirical breaking news website about our friend Collin discovering Canva and immediately believing he is a graphic designer.

**Live site:** https://collin-discovers-canva.vercel.app

## What is this?

A single-page parody news article featuring:

- Scrolling "BREAKING NEWS" ticker
- Live developing story timeline
- Fake quotes from friends, family, and Canva's support team
- Community reactions from concerned parties

## Run locally

Just open `index.html` in a browser. No build tools or dependencies needed.
